# Rasi Diri

Rasi Diri adalah aplikasi tes tipologi dan refleksi kepribadian berbasis React, TypeScript, dan Vite.

Aplikasi ini dirancang sebagai tes lokal di browser. Seluruh proses menjawab, scoring, dan hasil dilakukan di sisi frontend tanpa API key dan tanpa backend.

## Menjalankan Project
